# Archives

Older project artifacts kept for reference. Active scenarios live in `scenarios/` at the repo root.

## `test-scenarios/`

Original baseline test scenarios (scheduling, refills, insurance, etc.) from the initial project scaffold. Replaced by edge-case scenarios in `scenarios/` for the main test run.

To run an archived scenario, copy it into `scenarios/` or point `SCENARIOS_DIR` at this folder during development.

## `recordings/`

Move superseded call recordings here when you re-run scenarios and want to keep earlier takes. Match each file to its transcript in `archives/transcripts/` if you archive those as well.

Supported formats: `.mp3`, `.ogg`
